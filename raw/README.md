# Raw Layer — Primary Documents & Provenance Tags

Each entry is a short excerpt or pointer to a primary source, tagged with:

- **Provenance**: owner-mediated / multi-national shipping / archaeological / oral-community / administrative-legal / lab-derived
- **Enslaved voice?**: YES / PARTIAL / NO (or N/A)
- **Reliability note**: how the zero-weight mode treats this source

Do not treat any entry here as a numerical claim that survives maximal skepticism. Numbers appear only as they exist in the original document.
